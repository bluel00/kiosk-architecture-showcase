# Examples

이 폴더는 렌더러(React) 중심의 예시 코드를 담습니다. 모든 예시는 **실행 불가 스텁**이며, 빌드/실행 스크립트나 실제 하드웨어 코드를 포함하지 않습니다.

## 포함 파일
- `renderer-session.tsx`: 세션 상태 전이를 보여주는 렌더러 컴포넌트 스켈레톤.

## 작성 가이드
- IPC 호출, 상태 전이, 화면 전환과 같이 구조를 보여주는 부분에 집중합니다.
- 하드웨어 제어, 네트워크 호출, 브랜드/회사 식별자는 추가하지 않습니다.
- 타입 정의와 주석으로 계약을 명확히 표현하되, 프로토콜·경로·키와 같은 민감 요소는 생략합니다.
- 새 예시를 추가할 때는 어떤 계층(렌더러/프리로드/메인)에서 동작하는 코드인지 파일 상단에 명시합니다.
