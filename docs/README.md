# 문서 개요

이 폴더는 포트폴리오용 아키텍처 쇼케이스를 설명하는 문서를 모아둔 공간입니다. 실행 가능한 코드나 빌드 설정은 포함하지 않으며, **렌더러(React) 중심** 관점에서 전체 구성을 서술합니다.

## 포함 문서
- `architecture.md`: 전체 계층과 데이터 흐름 개요
- `flow.md`: 사용자 플로우와 상태 전이 규칙
- `ipc.md`: IPC 채널 계약 요약 및 타입 스텁
- `decisions/`: ADR 템플릿 및 기록 보관소

## 사용 가이드
- 하드웨어, 브랜드, 외부 서비스에 대한 구체 정보는 작성하지 않습니다.
- 예시는 타입/인터페이스/주석 수준에서만 다루며, 실제 프로토콜이나 경로를 명시하지 않습니다.
- 렌더러가 메인/프리로드와 협력하는 방식을 강조해 서술합니다.
- 새로운 주제를 추가할 때는 섹션별 링크를 업데이트하고, 필요 시 ADR을 작성합니다.
