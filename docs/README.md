# Docs 개요

이 폴더는 포트폴리오용 아키텍처 쇼케이스 문서를 모아둔 공간입니다. 실행 가능한 데모나 하드웨어 세부 구현은 포함하지 않고, 구조와 계약을 중심으로 설명합니다.

## 파일 안내
- `architecture.md`: 렌더러 중심의 전체 계층 구조와 설계 원칙 요약
- `flow.md`: 사용자 플로우와 상태 전이 규칙
- `ipc.md`: IPC 채널 계약과 타입 스텁
- `decisions/000-template.md`: ADR 작성용 템플릿

## 사용 가이드
- 예시 코드와 타입 스텁은 빌드 스크립트 없이도 읽기 전용으로 활용할 수 있습니다.
- 실제 회사명, 로고, 하드웨어 프로토콜 등 민감 정보는 추가하지 않습니다.
